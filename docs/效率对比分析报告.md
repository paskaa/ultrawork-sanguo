```

│           单一Agent                 │

**问题**:
- 通用模型不擅长所有任务

- 编程任务用推理模型成本高

    ┌───────────────────┼───────────────────┐
    │                   │                   │

┌─────────┐       ┌─────────┐       ┌─────────┐

│ GLM-5   │       │ Qwen    │       │ MiniMax │
│ 战略规划 │       │ 执行大将 │       │ 探索谋士 │
│ 设计任务 │       │ 开发任务 │       │ 搜索任务 │
└────┬────┘       └────┬────┘       └────┬────┘
     │                 │                 │
  ┌──┴──┐           ┌──┴──┐           ┌──┴──┐
  ▼     ▼           ▼     ▼           ▼     ▼
LuSu  HuangGai   GaoShun ChenDao  SimaShi SimaZhao
分析   执行       前端   后端       分析    整理
```

**优势**:
- 专业Agent处理专业任务
- 模型特性与任务匹配
- 成本与效果最优
