## 快速排查技巧

### 日志定位
```bash
# 查看最近错误
tail -100 logs/error.log

# 搜索关键词
grep "Exception" logs/*.log | tail -20
```

### 代码定位
```bash
# 搜索错误信息
Grep: "错误信息关键字"

# 查找相关方法
Grep: "public.*methodName"
```

### 数据库排查
```sql
-- 查看最近记录
SELECT * FROM table ORDER BY create_time DESC LIMIT 10;

-- 检查数据状态
SELECT status, COUNT(*) FROM table GROUP BY status;
```
