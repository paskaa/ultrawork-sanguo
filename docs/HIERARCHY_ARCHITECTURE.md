## 🔄 任务分发流程

### Step 1: 任务输入
```
用户输入任务 → 诸葛亮(主帅)接收
```

### Step 2: 意图分析
```
诸葛亮分析任务类型:
- 战略规划 → 周瑜
- 功能开发 → 赵云
- 代码探索 → 司马懿
- 代码审查 → 关羽
- 快速修复 → 张飞
- 系统监控 → 满宠
- 测试验证 → 徐庶
- 特殊任务 → 马超
- 文件操作 → 姜维
```

### Step 3: 任务分配
```
诸葛亮调用对应大都督，传递任务详情
```

### Step 4: 执行决策
```
大都督评估任务复杂度:
- 简单任务 → 独立执行
- 复杂任务 → 调用部将协助
```

### Step 5: 部将执行
```
大都督调用直属部将:
- 并行调用多个部将 (如: 高顺+陈到)
- 串行调用单个部将 (如: 仅高顺)
```

### Step 6: 结果汇总
```
部将 → 大都督 (汇报结果)
大都督 → 诸葛亮 (汇总上报)
诸葛亮 → 用户 (最终输出)
```

## 🚨 违规处理

### 检测到越级调用时

1. **立即终止**: 停止当前任务执行
2. **返回错误**: 返回层级错误信息
3. **提示正确路径**: 显示正确的调用链
4. **记录日志**: 记录违规行为供分析

### 错误信息示例

```
【层级错误】主帅(诸葛亮)只能调用大都督，禁止直接调用部将！
正确调用链: 诸葛亮 → [大都督] → [部将]
例如: 诸葛亮 → 赵云 → 高顺/陈到
```
